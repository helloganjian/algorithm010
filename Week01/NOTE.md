#第一周学习笔记

## 学习方法论

1. 五毒神掌，反复看视频，反复练习，职业化手法（手动狗头）
2. 画出思维导图，方便记忆和复习
3. 不要死磕，多看看别人的题解，理解大神的思路，写出简洁和fancy的代码
4. 掌握一些固定的解题模板，比如：

```javascript
/*** 暴力求解 ***/

		for (let k = j + 1; k < len; ++k) {
			...
```
